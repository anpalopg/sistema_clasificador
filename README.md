# Descripción del Proyecto

**Sistemas Clasificadores** es un proyecto orientado a la planificación y desarrollo de un sistema que permita organizar y clasificar información mediante una estructura de trabajo organizada.

El proyecto utiliza herramientas de gestión como:

- WBS (Work Breakdown Structure)
- Matriz RACI
- Cronograma del proyecto
- Presupuesto del proyecto
- Control de actividades

Estas herramientas permiten una planificación estructurada del desarrollo del sistema.

---

# Objetivo del Proyecto

Planificar y estructurar el desarrollo del proyecto **Sistemas Clasificadores** mediante herramientas de gestión que permitan:

- Definir las tareas necesarias para el desarrollo
- Organizar el trabajo del equipo
- Controlar tiempos y recursos
- Establecer responsabilidades
- Realizar seguimiento del progreso del proyecto

# Matriz RACI

| Rol | Significado |
|----|----|
| R | Responsible – Responsable de ejecutar la tarea |
| A | Accountable – Responsable final |
| C | Consulted – Persona consultada |
| I | Informed – Persona informada |

La matriz RACI permite definir claramente las responsabilidades dentro del equipo.

# Estructura del Proyecto (WBS)

El proyecto se divide en las siguientes fases principales:

1. Gestión del proyecto  
2. Planificación  
3. Desarrollo  
4. Seguimiento  
5. Cierre  

Esta estructura permite organizar el trabajo en componentes más pequeños y manejables.

# Cronograma del Proyecto

El cronograma permite organizar las actividades del proyecto en función del tiempo.

Incluye:

- Fecha de inicio
- Fecha de finalización
- Duración de tareas
- Dependencias entre actividades

Esto permite llevar un control del progreso del proyecto.

---

# Gestión de Actividades

Las actividades del proyecto se registran mediante una lista de tareas que incluye:

- Nombre de la actividad
- Responsable
- Estado de la actividad
- Porcentaje de avance

Estados posibles de las tareas:

- No iniciado
- En progreso
- Completo
- Atrasado
- En espera

# Presupuesto del Proyecto

El presupuesto del proyecto incluye estimaciones relacionadas con:

Esto permite administrar los recursos financieros durante el desarrollo.
